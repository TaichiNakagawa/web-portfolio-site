Technical Support & Maintenance

Overview

IT領域におけるシステム開発をメインで展開する企業の公式サイト立ち上げにおいて、技術的な土台作りと運用サポートを担当しました。

初期の課題として、多様なデバイスからの閲覧を支えるレスポンシブ対応、および専門知識を持つ担当者がスムーズに情報を発信できる環境の構築が求められていました。

Core Support

01. レスポンシブ化の完遂

既存のデスクトップ向けレイアウトを維持しつつ、スマートフォンやタブレットからの視認性を最適化しました。ITエンジニアが現場などの外出先からでも情報をキャッチできるよう、読みやすさに配慮した設計を行いました。

02. ローカル開発環境の構築

Local環境をセットアップ。本番環境に影響を与えずに検証できるフローを構築しました。

03. 運用マニュアル作成とレクチャー

「技術を渡して終わり」ではなく、継続的な運用を可能にするためにカスタマイズされたマニュアルを作成。電話対応や面談を通じ、技術的な不明点を解消する手厚い導入サポートを提供しました。

Role

Maintenance Support / Developer

Tech Stack

HTML5 / CSS3 JavaScript Documentation WordPress

Challenge

専門性の高いサービスにふさわしい誠実なデザインを維持しつつ、技術的な不具合（画像表示のバグ等）を徹底的に排除しました。
